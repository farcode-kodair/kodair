Alpha V1.9 (2/27/22)
The Kodfeed Update: A new Kodfeed app has been added which not only lets you read rss feeds, but also many other types of feeds including twitter and reddit. Also, a temporary app called CryptoColors has been added to test blockchain powered art features (feel free to try it for yourself).
Alpha V1.8 (2/26/22)
The Decenart Update: If you have some crypto laying around and don't know what to do with it, this update is for you. A new app has been added called Decenart, which allows you to draw on a public canvas for all to see for the small price of some crypto.
Alpha V1.7 (2/25/22)
The Kodbit Update: If you like audio and you like to code, then you will love Kodbit! Kodbit is a sound script editor that lets you create sounds with code. Even if you don't know how to make sounds with code, just check out some of the examples and play around with the numbers in the code to create new sounds.
Alpha V1.6 (2/24/22)
The Kodoshop Update: To go along with our recent video editor update, we are glad to introduce Kodoshop! Kodoshop is your brand new photo editor. Enjoy creating amazing photos, images, and animation with this amazing app.
Alpha V1.5 (2/23/22)
The Kodcut Update: For a while you have been able to watch videos on this site, but now you will also be able to create them. Introducing Kodcut! A video editor built right into your browser. Enjoy creating videos with this new app.
Alpha V1.4 (2/22/22)
The KodRuffle Update: A new app called KodRuffle has been added! YThis app lets you play all of your favorite flash (.swf) games, and all you have to do is provide the swf file. Also, the Kodsly Password Manager has been updated.
Alpha V1.3 (2/21/22)
The Kordle Update: A new Kordle app has been added! You can now play this fun word game and test your word guessing skills. Also, the FlashCards app has been re-designed to make it look cleaner, and some new commands have been added to Kodcom.
Alpha V1.2 (2/20/22)
The KodTube Update: A new KodTube channel has been added to the Kodscan app. Now you can search for and watch all of your favorite video from YouTube, Twitch, DailyMotion, and more by simply typing pasting the video link or by searching for it. However, that's not all. The Kodscan app now has a ThetaTv and PlutoTv channel for viewing your favorite live streams and live television. Finally, if you use the Kodscan app long enough you may find a surprise show up in the corner of your screen.
Alpha V1.1 (2/2/22)
The Developer Update: A new Kodcom terminal app and KodEditor HTML development app have been added. The KodCube Minecraft clone has been updated and renamed to Kodcraft. Some other small cosmetics changes have been added as well to make you experience that much better. Last but not least, Kodair's source code is now available on Github!
Alpha V1.0 (12/22/21)
The Utilities Update: The time and date have been moved to that app bar. Clicking on the time opens up a new clock app. Also, a new weather app has been added that emulates the old cable tv style weather channel. Overall, Kodair can now help you prepare for your day with clocks, calendars, and weather.
Alpha V0.9 (12/11/21)
The Security Update: More password managers have been added in order to provide the user with more choices. Also, Kodits and the side pannel have been temporarily removed as they serve no puurpose at the moment.
Alpha V0.8 (12/9/21)
The Password Update: A new password manager app has been added. It uses stateless password generation powered by Unstated to generate secure passwords (this is only for demonstration purposes at the moment, so don't rely on it for your passwords). Also, a new option to purchase crypto has been added to the Kodex app.
Alpha V0.7 (12/8/21)
The Exchange Update: The Kodex app has been added. Within the app you can exchange your cryptocurrency for other cryptocurrencies. At the moment, only exchanges are available, but other features are in the works.
Alpha V0.6 (12/7/21)
The Entertainment Update: The Radio and Television apps have been merged together to create an all in one media player and entertainment app.
Alpha V0.5 (12/6/21)
The Drawing Update: A new drawing app has been added to the site. You can now draw and animate whatever you want. Also, credit has been given to the creator of this site at the bottom of the info section.
Alpha V0.4 (12/5/21)
The Scrolling And Size Update: Many of the clunky scrolling bars were removed in order to make the site cleaner and less cluttered, and the sizes of some of the elements were altered to make for a better experience. Also, the scrolling issue in the updates and info section (this section) has been fixed.
Alpha V0.3 (12/4/21)
The Education Update: A new EDU app has been added. This app will contain educational content, theories, and opinions. Also, some unnessesary stuff was removed from the website and some of the scrollbars have been removed in order to make the site look cleaner.
Alpha V0.2 (12/3/21)
The RSS Update: Replaced the ebook reader with an RSS reader until the ebook reader can be fixed. The RSS reader has some touching up to do, but it gets the job done. Also, the welcome page has been decluttered as some of the information on that page was outdated. So, enjoy the site and add your favorite RSS feeds!
Alpha V0.1 (12/2/21)
The First Official Release Version: Welcome to Kodair! Take a look around. Things have been polished, new apps have been added, and more. However, there are hundreds more apps in the works and much more polishing to be done. So, stick around, have some fun, and share this site with others!
Alpha V0.0 (12/1/21)
The Very Beggining: Everything up to this version was just testing, now it's time to make this website a reality!
